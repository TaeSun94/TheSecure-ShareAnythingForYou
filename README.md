## 개요

> 언택트 시대가 나온 뒤 여행지에서 대형 숙소 이용을 꺼려하는 사용자가 증가함에 따라 하우스 쉐어링이 증가할 것을 예측하여 중복 예약, 예약 번복 등을 막기위하여 블록체인을 도입하여 서비스 개발

## 기능

> 쉐어링 하우스 등록, 쉐어링 신청, 블록체인 지갑 개설, 코인 환전

## 유사 서비스

> 유사 서비스 : Airbnb
>
> 차이점 : 예약 위변조 불가능, 통용화폐 사용으로 수수료 절감
